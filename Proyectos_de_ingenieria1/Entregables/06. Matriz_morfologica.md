## 6. 🧩 **Matriz Morfológica** (Responsable: José Jayro Pereira)

La matriz morfológica descompone el sistema WILLAY en sus funciones principales y explora las posibles soluciones técnicas para cada una.

### **Tabla 1: Matriz Morfológica del Sistema WILLAY**

| Función Principal | Opción 1 | Opción 2 | Opción 3 | Opción 4 | **Seleccionada** |
|------------------|----------|----------|----------|----------|------------------|
| **Captura de Datos Climáticos** | Estación meteorológica física | Sentinel-1 SAR + Sentinel-2 Óptico | API SENAMHI | Drones con sensores | **Satélites (SAR+Óptico)** |
| **Procesamiento de IA** | Redes Neuronales CNN | **BiLSTM Bidireccional** | Random Forest | Regresión Lineal | **BiLSTM** |
| **Transmisión de Datos** | WiFi directo | **Bluetooth LE + MQTT** | LoRaWAN | GSM/SMS | **BLE + MQTT** |
| **Alimentación Energética** | Batería desechable | **Li-ion 18650 recargable** | Panel solar directo | Conexión a red | **Li-ion + Solar opcional** |
| **Interfaz de Usuario** | Web desktop | **App móvil + SMS** | Dashboard web | Llamadas automáticas | **App + SMS** |
| **Almacenamiento de Datos** | Local (SD card) | **Firebase/Cloud** | Servidor propio | Blockchain | **Firebase/Cloud** |
| **Validación en Campo** | Sin validación | **Sensores IoT (ESP32)** | Muestreo manual | Estaciones oficiales | **ESP32 + DHT22/BME280** |
| **Idioma de Alertas** | Solo español | **Español + Quechua** | Solo Quechua | Inglés + Español | **Español + Quechua** |

### **Justificación de Selección:**

```
✅ Satélites: Penetran nubosidad andina (SAR) + resolución espectral (Óptico)
✅ BiLSTM: Captura patrones temporales bidireccionales en series climáticas
✅ BLE + MQTT: Bajo consumo, eficiente para IoT agrícola
✅ Li-ion: Recargable, sostenible, disponible en mercado local
✅ App + SMS: Accesible para zonas de baja conectividad
✅ Cloud: Escalable, accesible desde cualquier dispositivo
✅ ESP32: Económico, programable, disponible en Perú (S/ 25-40)
✅ Bilingüe: Inclusivo para comunidades andinas nativas
```
