## 6. 🧩 **Matriz Morfológica** 

Sistema IoT de Monitoreo Térmico y Alerta Temprana para Entornos Agrícolas Altoandinos (MVP)

  ## A.  INTRODUCION: 
  El proyecto WILLAY propone el desarrollo de un sistema IoT orientado al monitoreo climático local y generación de alertas tempranas ante posibles eventos de helada en zonas agrícolas altoandinas.

Debido al alcance académico y a las limitaciones de recursos, el proyecto se plantea como un MVP (Minimum Viable Product), enfocado en validar la arquitectura funcional del sistema, la adquisición de datos climáticos y la comunicación básica de alertas.

 ## B.  OBJETIVO:
 El prototipo busca:

- Medir variables climáticas básicas en campo
- Detectar variaciones térmicas asociadas a riesgo de helada
- Generar alertas locales
- Transmitir y almacenar datos climáticos
- Construir una base de datos para futuras etapas de modelado predictivo

 ## E. Criterios de selección

La selección de alternativas para el prototipo se realizó considerando los siguientes criterios:

 - Viabilidad técnica

Se priorizaron soluciones compatibles con el nivel de desarrollo actual del proyecto y con disponibilidad de implementación inmediata.

 - Viabilidad económica

Se seleccionaron componentes de bajo costo debido a las limitaciones presupuestarias del equipo.
 
- Facilidad de integración

Se priorizaron tecnologías ampliamente documentadas y compatibles entre sí.

- Funcionamiento en campo

El sistema debe ser capaz de operar en pruebas reales y validar el flujo básico de adquisición y transmisión de datos.

## F. **Justificación de Selección:**

La configuración seleccionada prioriza:

  - bajo costo de implementación
  - facilidad de integración
  - funcionamiento autónomo
  - estabilidad para demostración funcional
  - compatibilidad entre componentes
  - rapidez de desarrollo para validación académica

La incorporación de batería recargable permite mantener coherencia con el objetivo del proyecto, orientado a posibles aplicaciones en zonas agrícolas con acceso limitado a infraestructura eléctrica.

El sistema se centra en la adquisición y transmisión básica de datos climáticos, permitiendo validar la arquitectura general antes de incorporar funciones más complejas como análisis predictivo avanzado o integración satelital.

La arquitectura también permite escalabilidad futura hacia:

  - modelos predictivos
  - integración de datos remotos
  - comunicación LoRa
  - alertas móviles avanzadas

sin modificar completamente la estructura base del sistema.


## Aclaraciones técnicas importantes

- Transferencia de datos: Se refiere al destino de la información (local, teléfono, nube).
- Almacenamiento: Se realiza exclusivamente en la nube (Firebase, AWS u otros).
- Modelado: No se fija un modelo único; se evaluarán múltiples enfoques.
- Energía: Se utilizará bateria.


La matriz morfológica permitió estructurar de manera organizada el diseño funcional del sistema WILLAY, facilitando la evaluación de múltiples alternativas tecnológicas y la selección de una configuración coherente con los objetivos y limitaciones del proyecto.

La solución propuesta constituye una base escalable para futuras aplicaciones de monitoreo climático inteligente en agricultura altoandina.


El prototipo desarrollado no solo valida el concepto de monitoreo climático, sino que establece una base sólida para la implementación futura de modelos predictivos aplicados a la agricultura andina.
